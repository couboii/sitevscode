PROGRAMADOR FRONT-END

Front End development é o profissional de tecnologia que atua no desenvolvimento de sites e soluções que vão interagir diretamente com o usuário. Isso quer dizer que ao utilizar um determinado ambiente digital, o usuário vai interagir com os recursos e funcionalidades. Alguns exemplos são o layout do site, botões, áreas específicas etc. Tudo que está ao alcance e na interface que o usuário está tendo contato direto, ou seja, podendo interagir, é feito pelo Front End. O desenvolvedor cria um código, geralmente, utilizando as linguagens em HTML, CSS e JavaScript. Porém, algumas pessoas podem confundir o trabalho do Front End com o do designer, pois ambos atuam na interface.
